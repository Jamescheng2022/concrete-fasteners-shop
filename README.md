# ProConnect Fasteners - Precast Concrete Connectors Website

A complete B2B foreign trade website for selling precast concrete fasteners globally.

## Pages
- `index.html` — Homepage with hero, featured products, and stats
- `products.html` — Full product catalog with categories and pricing
- `about.html` — Company story, certifications, and team
- `contact.html` — Inquiry form with FAQ

## Deployment on Vercel

### Step 1: Upload to GitHub
1. Go to [github.com](https://github.com) and create a new repository
2. Upload all files from this folder

### Step 2: Deploy to Vercel
1. Go to [vercel.com](https://vercel.com)
2. Click "Add New Project"
3. Import your GitHub repository
4. Vercel will auto-detect the static site — click **Deploy**
5. Done! Your site is live.

## Customization
- Replace placeholder contact info (email, phone, WeChat)
- Replace company name and address
- Update pricing as needed
- Add real product images to `/images/` folder

## Tech Stack
- Pure HTML + Tailwind CSS (CDN)
- No build step required
- Zero dependencies
